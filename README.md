# Run them on your computer!
The solutions are in Python and maybe sometimes in OCaml, so you can run them with
```sh
python3 day1.py
```
or 
```sh
ocaml day1.ml
```

> __Note:__ These programs assume you have your input file named "input.txt" in the same folder as the code.
